    ```typescript
    const { propertyId } = useLocalSearchParams<{ propertyId: string }>();
    ```
    Query `supabase.from('tenants').select('id').eq('property_id', propertyId).eq('user_id', user.id).single()` to get tenantId on mount.

    Form layout (per UI-SPEC):
    1. Title: single-line TextInput (react-native-paper), label "Title", placeholder "e.g. Leaking kitchen tap"
    2. Description: multi-line TextInput, label "Description", minHeight 100, numberOfLines 4, placeholder "Describe the issue in detail"
    3. Priority: horizontal segmented control — three TouchableOpacity cells in a row, equal width (flex 1), 48px height, borderRadius 10, border colors.border. Active: backgroundColor colors.primary, text colors.textOnPrimary at 14px/700. Inactive: backgroundColor colors.surface, text colors.textSecondary at 14px/400. Labels: "Low" | "Normal" | "Urgent". Default "Normal" selected (per D-04).
    4. Photos: `<MaintenancePhotoUploader photos={photos} onPhotosChange={setPhotos} maxPhotos={5} />`
    5. Submit button: full-width button, label "Submit Request", disabled until title.trim() AND description.trim() are non-empty. Show ActivityIndicator while submitting.

    Submit flow (CRITICAL ORDER — per Research pitfall #1):
    1. INSERT request row with `photo_paths: '{}'` (empty initially):
       ```typescript
       const { data: inserted, error } = await supabase
         .from('maintenance_requests')
         .insert({
           property_id: propertyId,
           tenant_id: tenantId,
           title: title.trim(),
           description: description.trim(),
           priority,
           photo_paths: [],
         })
         .select('id')
         .single();
       ```
    2. INSERT initial status log (per Research pitfall #3):
       ```typescript
       await supabase.from('maintenance_status_logs').insert({
         request_id: inserted.id,
         changed_by: user.id,
         from_status: null,
         to_status: 'open',
       });
       ```
    3. Upload photos sequentially (if any):
       ```typescript
       import * as FileSystem from 'expo-file-system/legacy';
       import { MAINTENANCE_PHOTOS_BUCKET } from '@/constants/config';

       const uploadedPaths: string[] = [];
       for (const photo of photos) {
         const base64 = await FileSystem.readAsStringAsync(photo.uri, { encoding: FileSystem.EncodingType.Base64 });
         const { decode } = await import('base64-arraybuffer');
         const buffer = decode(base64);
         const storagePath = getMaintenancePhotoPath(propertyId, inserted.id, 'jpg');
         const { error: uploadError } = await supabase.storage
           .from(MAINTENANCE_PHOTOS_BUCKET)
           .upload(storagePath, buffer, { contentType: 'image/jpeg', upsert: false });
         if (!uploadError) uploadedPaths.push(storagePath);
       }
       ```
    4. UPDATE photo_paths if any uploaded:
       ```typescript
       if (uploadedPaths.length > 0) {
         await supabase.from('maintenance_requests')
           .update({ photo_paths: uploadedPaths })
           .eq('id', inserted.id);
       }
       ```
    5. Send push notification to landlord (MAINT-02, per Research pattern 4):
       ```typescript
       const { data: property } = await supabase
         .from('properties')
         .select('owner_id')
         .eq('id', propertyId)
         .single();
       const { data: owner } = await supabase
         .from('users')
         .select('push_token')
         .eq('id', property.owner_id)
         .single();
       if (owner?.push_token) {
         await supabase.functions.invoke('send-push', {
           body: {
             messages: [{
               token: owner.push_token,
               title: 'New Maintenance Request',
               body: `${user.full_name ?? 'A tenant'}: ${title.trim().substring(0, 80)}`,
               data: { type: 'maintenance_new', requestId: inserted.id },
             }],
           },
         });
       }
       ```
    6. Show success toast, navigate back:
       ```typescript
       showToast('Maintenance request submitted!', 'success');
       router.back();
       ```
    7. If photo upload partially fails, show warning toast: "Request submitted, but one or more photos failed to upload. You can resubmit with photos."
    8. If INSERT fails, show error toast: "Could not submit your request. Please check your connection and try again."
