**DocumentUploader.tsx:**

Create `components/DocumentUploader.tsx`:

Props interface:
```typescript
interface DocumentUploaderProps {
  visible: boolean;
  onClose: () => void;
  onUploadComplete: () => void;
  propertyId: string;
  tenantId: string | null; // null = property-wide
}
```

State: `asset` (picked file or null), `name` (string, pre-filled from filename), `category` (DocumentCategory, default 'other'), `isUploading` (boolean).

Upload flow per UI-SPEC Interaction Contracts:
1. Render as `Modal` (animationType 'slide', transparent false).
2. Surface with borderRadius 16 at top, padding 24.
3. Title: "Upload Document" (17px, weight 700).
4. File preview area: if asset picked and image MIME → show `Image` preview (120px height); if PDF/Word → show filename badge.
5. "Choose File" button (`mode: 'outlined'`, icon `'file-plus'`): calls `DocumentPicker.getDocumentAsync({ type: ['application/pdf', 'image/jpeg', 'image/png', 'application/msword', 'application/vnd.openxmlformats-officedocument.wordprocessingml.document'], copyToCacheDirectory: true, multiple: false })`.
6. On file selected:
   - Check `asset.size > 10485760` → show toast "File exceeds 10 MB limit. Please choose a smaller file." and return.
   - Set `asset` state, pre-fill `name` with `asset.name` (strip extension from display name).
7. TextInput: label "Document Name", placeholder "e.g. Lease Agreement Jan 2026", value bound to `name` state.
8. Category picker: render 5 `TouchableOpacity` chips (from `ALL_CATEGORIES` with `CATEGORY_LABELS`), highlight selected.
9. Button row:
   - Cancel: `mode: 'text'`, `colors.textSecondary`, calls `onClose()`.
   - Upload: `GradientButton`, disabled when `!asset || !name.trim() || isUploading`, loading prop during upload.
10. On Upload press:
    - Set `isUploading = true`.
    - Generate storage path: `getDocumentStoragePath(propertyId, tenantId, getExtFromFilename(asset.name))`.
    - Call `uploadDocument(asset, storagePath)`.
    - Insert DB row: `supabase.from('documents').insert({ property_id: propertyId, tenant_id: tenantId, uploader_id: user.id, name: name.trim(), category, storage_path: storagePath, mime_type: asset.mimeType, file_size: asset.size })`.
    - On success: toast "Document uploaded", call `onUploadComplete()`, call `onClose()`.
    - On failure: toast "Upload failed. Please try again.", keep modal open.
    - Set `isUploading = false`.

Import `* as DocumentPicker from 'expo-document-picker'`.

**DocumentViewer.tsx:**

Create `components/DocumentViewer.tsx`:

Props interface:
```typescript
interface DocumentViewerProps {
  visible: boolean;
  document: Document | null;
  onClose: () => void;
}
```

State: `signedUrl` (string | null), `isLoading` (boolean), `error` (string | null), `isSharing` (boolean).

On `visible` change + document not null:
- Set `isLoading = true`, `error = null`.
- Call `getSignedUrl(document.storage_path)`.
- On success: set `signedUrl`.
- On failure: set `error`.
- Set `isLoading = false`.

Render as `Modal` (animationType 'slide', presentationStyle 'fullScreen'):
- `SafeAreaView` wrapping everything.
- Top bar (56px height, 16px horizontal padding):
  - Left: Close button (`'close'` icon, 44x44 touch target, `colors.textPrimary`, accessibilityLabel "Close document viewer"), calls `onClose()`.
  - Center: Document name (16px, weight 700, numberOfLines 1).
  - Right: empty spacer View for centering.
- Content area (flex: 1):
  - If `isLoading`: `ActivityIndicator` centered, color `colors.primary`.
  - If `error`: centered text "Could not load document. Tap to retry." with `TouchableOpacity` wrapping to retry `getSignedUrl`.
  - If image MIME (`isImageMime(document.mime_type)`): render `Image` with `resizeMode: 'contain'`, full-size. For pinch-to-zoom, wrap in `ScrollView` with `maximumZoomScale={3}` and `minimumZoomScale={1}`.
  - If PDF or Word: render `WebView` with `source={{ uri: getViewerUrl(signedUrl, document.mime_type) }}`. Show `ActivityIndicator` overlay while WebView is loading (`onLoadStart`/`onLoadEnd`).
- Bottom bar (72px height, 16px padding, top border `colors.divider`):
  - Share button (full width, `mode: 'outlined'`, icon `'share-variant'`, label "Share Document", loading prop during share).
  - On press: call `shareDocument(document, signedUrl)`. On error: toast "Could not share file."

Import `WebView` from `react-native-webview`.

Reset state (signedUrl, error, isLoading) when modal closes or document changes.
