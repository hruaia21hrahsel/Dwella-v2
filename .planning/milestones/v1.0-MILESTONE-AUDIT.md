## Integration Findings

### INT-01 — ESLint scope gap (High)
ESLint `files` array excludes `supabase/functions/**`. 8 `as any` casts in Edge Functions are invisible to the lint gate. Affects TS-02, TS-03. Note: Edge Functions run in Deno — may need separate parser config.

### INT-02 — WhatsApp HMAC silent bypass (Medium)
`validateMetaSignature` returns `true` when `WHATSAPP_APP_SECRET` env var is absent. A misconfigured production deployment silently accepts all webhook requests. Affects SEC-05.

### INT-03 — Dead fallback code (Low)
`lib/supabase.ts` retains `|| 'placeholder'` fallbacks unreachable after `requireEnv`. Non-functional but misleading. Affects CLIENT-02.

### INT-04 — iOS store URL placeholder (High — pre-launch)
`components/UpdateGate.tsx` line 8 has literal `[APP_ID]` in the iOS App Store URL. Forced-update redirect fails on iOS until replaced. Already tracked in MEMORY.md pre-launch checklist. Affects LAUNCH-04.
