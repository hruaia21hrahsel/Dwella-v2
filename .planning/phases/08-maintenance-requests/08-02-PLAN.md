    - `useCallback` fetch function that queries `supabase.from('maintenance_requests').select('*').eq('property_id', propertyId).eq('is_archived', false).order('created_at', { ascending: false })`

Cleanup: `supabase.removeChannel(channel)`. Depends on [propertyId, fetch].
    - Return { requests, isLoading, error, refresh: fetch }
