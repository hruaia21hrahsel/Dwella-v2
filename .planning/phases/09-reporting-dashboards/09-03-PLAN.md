**TimeControlBar.tsx (per D-06, D-07, D-08, D-09):**

Props:
```typescript
interface TimeControlBarProps {
  period: TimePeriod;
  onPeriodChange: (period: TimePeriod) => void;
}
```

Implementation:
- **Year row:** Left chevron (`chevron-left`) + year text (16px, weight 700) + right chevron (`chevron-right`). Min year = 2000, max year = current year. Chevrons are `TouchableOpacity` with `hitSlop={{ top: 10, bottom: 10, left: 10, right: 10 }}`. All touch targets minimum 44px tall.
- **Granularity row:** 3 chips in a horizontal `View` with gap 8px:
  - Labels: "Yearly", "Quarterly", "Monthly"
  - Active chip: `backgroundColor: colors.primarySoft`, `borderColor: colors.primary`, `borderWidth: 1`, text color `colors.primary`
  - Inactive chip: `backgroundColor: colors.surface`, `borderColor: colors.border`, `borderWidth: 1`, text color `colors.textSecondary`
  - Chip text: 13px, weight 400. Chip height: minimum 44px (accessibility).
  - On press: call `onPeriodChange` with updated granularity. When switching to quarterly, default quarter = current quarter. When switching to monthly, default month = current month.
- **Period selector row (conditional):**
  - When granularity === 'quarterly': Show 4 chips — "Q1", "Q2", "Q3", "Q4". Same active/inactive styling.
  - When granularity === 'monthly': Show 12 chips in a horizontal `ScrollView` — "Jan" through "Dec" (use `getMonthName(m).slice(0, 3)`). Future months in the current year: text color `textDisabled` but still tappable.
  - When granularity === 'yearly': No period selector row rendered.
- All rows: horizontal padding 0 (parent screen handles padding). Vertical gap between rows: 8px.
- Use `useTheme()` for all colors.
