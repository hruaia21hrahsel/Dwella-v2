## 2. Description of Service

Dwella is a mobile application that helps landlords and tenants manage rental properties, track rent payments, and communicate via an AI-powered bot. The App serves two user roles:

- **Landlords** can create properties, add tenants, log payments, send reminders, generate receipts, and use AI-assisted rental management via the Telegram bot.
- **Tenants** can view their rental details, payment history, and receive notifications when linked to a property by their landlord via an invite.
- A single user account may function as both landlord and tenant simultaneously, depending on property ownership and invite acceptance.
